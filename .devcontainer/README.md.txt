# Glassdoor Streamlit Scraper

## Objectif
Extraire facilement les avis Glassdoor pour WeFiiT via une interface Streamlit simple.

## Utilisation

1. Charge ton fichier `glassdoor_credentials.txt` contenant sur 2 lignes :
    ```
    email=ton.email@exemple.com
    password=tonMotDePasse
    ```
2. Clique sur "Lancer le scraping"
3. Télécharge le fichier CSV avec les avis extraits

## Déploiement Streamlit

- Déploie ce repo sur [Streamlit Cloud](https://streamlit.io/cloud)
- Sélectionne `app.py` comme script principal
